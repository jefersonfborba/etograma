## Story
**As a** pesquisador usando o Etograma no Android,
**I want** exportar meus dados em CSV normalmente,
**so that** eu possa analisar os registros etológicos no computador, mesmo com o app instalado como APK nativo.

## Acceptance Criteria

1. A exportação de CSV funciona no app Android (APK instalado via sideloading)
2. O fluxo de exportação usa `@capacitor/filesystem` para salvar o arquivo e `@capacitor/share` para abrir o compartilhamento nativo do Android
3. No ambiente web (browser), o comportamento original de download via `blob:` URL é preservado
4. O arquivo CSV exportado tem o nome correto (ex: `etograma-{projeto}-{data}.csv`)
5. Após compartilhar, o usuário pode salvar o arquivo em Google Drive, WhatsApp, email, etc.
6. Se `@capacitor/share` não estiver disponível (web), o fallback para `blob:` é automático

## Dev Notes

### Padrão de detecção de ambiente

```javascript
function isNativeApp() {
  return !!(window.Capacitor && window.Capacitor.isNativePlatform());
}
```

### Implementação de exportação nativa

```javascript
import { Filesystem, Directory } from '@capacitor/filesystem';
import { Share } from '@capacitor/share';

async function exportCsvNative(csvContent, fileName) {
  // Salva no cache temporário
  const result = await Filesystem.writeFile({
    path: fileName,
    data: csvContent,
    directory: Directory.Cache,
    encoding: 'utf8'
  });

  // Abre o share sheet nativo do Android
  await Share.share({
    title: 'Exportar dados etograma',
    url: result.uri,
  });
}
```

### Função adaptada (padrão recomendado)

```javascript
async function exportarCSV() {
  const csvContent = gerarCSV(); // função existente
  const fileName = `etograma-${nomeProjetoAtual}-${formatarData()}.csv`;

  if (isNativeApp()) {
    await exportCsvNative(csvContent, fileName);
  } else {
    // Comportamento web original — preservado
    const blob = new Blob([csvContent], { type: 'text/csv;charset=utf-8;' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = fileName;
    a.click();
    URL.revokeObjectURL(url);
  }
}
```
